INTRO

This Javascript User Interface, given a Price Amount and Paid Amount, based on what is present in the Cash-Drawer can: Calculate what change give back, divided in each unit, Update the Cash-Drawer, depending on the transactions done, Keep Track of all the Transactions, Inform the user either there are some errors or the cash is impossible to cover.

In this DEMO version, the Drawer starts with an initial amount. It will be editable in the Full version

VIDEO DEMO:

HOW-TO USE:

Insert a random price in the Price field

Then insert an amount paid, Please split by clicking each unit-button:

Example:
The Customer gives you 80$ => Ex. n.4 $20 => Click 4 times 20$ button

Then click the Calculate button, to get the Cash to give back.

Clicking the Drawer button in CHECK THE DRAWER area, you can check what is currently present in the Drawer

Clicking the Report Page button, user can go to the report page, reporting all the valid transactions.

The program will generate a Pop-up, informing about the Price, The Amount paid and what give back (divided in each unit)

The algorithm can generate an error if you, by mistake, try to insert, for example, an amount of money paid less than then the price

The program will also warns User if, by mistake, he forgets to fill the price, or filled with errors, or forget to fille the paid amount.

The program will also warns User if the amount to give back is impossible to cover, with the amount present in the Drawer

Clicking Drawer button, the program will show the drawer, updated with the transactions done.

User can find a report of Each valid transactions done in the Report Page.

Clicking the Save the Report button, you can download a .JSON file with all the transactions done.
